# إدارة المهام

مشروع ويب بسيط لإدارة المهام اليومية، يسمح للمستخدمين بإضافة مهام جديدة، تعديلها، حذفها، وعرضها، بالإضافة إلى تغيير حالة المهام (مكتملة أو غير مكتملة).  
المشروع يستخدم **PHP وMySQL** مع واجهة بسيطة باستخدام **HTML وCSS**، ويعمل على سيرفر محلي مثل **XAMPP**.

---

## المتطلبات

- **PHP ≥ 8.2** (تم اختبار المشروع على PHP 8.2.12)
- **MySQL**  
- **سيرفر محلي** مثل XAMPP
- **متصفح حديث** (Chrome, Firefox, Edge)
- **محرر نصوص** مثل VS Code
- (اختياري) استيراد قاعدة البيانات عبر phpMyAdmin إذا كان هناك ملف SQL جاهز

---

## التثبيت والتشغيل

لتشغيل المشروع على جهازك المحلي:

1. تأكد من تشغيل **XAMPP** وتشغيل Apache وMySQL.
2. انسخ جميع ملفات المشروع إلى مجلد السيرفر المحلي، عادةً:
3. افتح **phpMyAdmin** من XAMPP.
4. أنشئ قاعدة بيانات جديدة باسم:
5. استورد قاعدة البيانات (إذا عندك ملف SQL جاهز):
- اضغط على قاعدة البيانات `taskmanagement`.
- اختر **Import** ثم اختر الملف `.sql` المرتبط بالمشروع.
6. افتح المتصفح واكتب:
7. سجّل دخولك أو سجّل مستخدم جديد لتبدأ باستخدام المشروع.

> **ملاحظة:** إذا لم يكن لديك ملف SQL جاهز، يمكنك تعديل `config.php` لإنشاء الجداول مباشرة، لكن يفضل دائمًا استيراد قاعدة جاهزة لتجنب أي مشاكل.

---

## طريقة الاستخدام

- **تسجيل مستخدم جديد:** افتح `register.html` وأدخل البيانات المطلوبة.  
- **تسجيل الدخول:** افتح `login.php` وأدخل بيانات المستخدم.  
- **إضافة مهمة:** بعد تسجيل الدخول، استخدم النموذج لإضافة مهمة جديدة.  
- **تعديل مهمة:** اضغط على زر التعديل بجانب المهمة لتغيير التفاصيل.  
- **حذف مهمة:** اضغط على زر الحذف بجانب المهمة لإزالتها.  
- **تغيير حالة المهمة:** اضغط على زر الحالة لتبديل المهمة بين مكتملة وغير مكتملة.
